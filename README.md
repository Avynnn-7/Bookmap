# Bookmap Clone

This is my project for a Bookmap clone. It has a C++ backend for the engine, a Node.js server, and a React frontend.

## Components
* `cpp-engine/`: The core engine that processes market data.
* `server/`: Node.js server to handle WebSocket connections.
* `ui/`: React frontend to show the order book and heatmap.

## How to run it
First start the C++ engine. Then run the Node.js server. Then start the React app. I will put more instructions in the folders.
